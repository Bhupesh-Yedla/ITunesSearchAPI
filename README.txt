ITunesSearchAPI

This project contains automated tests for Itunes Search API functionality. It utilizes Rest Assured, TestNg testing frameworks and Maven, Extent Reports for executing tests and generating reports.

Prerequisites
Before running the tests, ensure you have the following installed:

Java Development Kit (JDK) 11 or higher
Apache Maven 3.6.0 or higher

Getting Started
Follow these steps to run the automated tests:

Clone Repository: Clone the project repository from GitHub:

git clone https://github.com/Bhupesh-Yedla/ITunesSearchAPI.git
Navigate to Project Directory: Change into the project directory:

cd ITunesSearchAPI project
Build Project: Build the project using Maven:

mvn clean install
Running Tests
To execute the automated tests, use the following command:

mvn test -Dtest=classname#testcasenme 
Test Results
After running the tests, test reports will be generated in the test-output/ directory. In addition, detailed test reports can be found in the target/surefire-reports.
